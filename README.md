# Terrier Inventory Management

This app is now backed by a real Node.js API and SQLite database.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. Open `http://localhost:3000`

## Login

- Admin account: `admin` / `tcs123`
- Users can sign in with any unique username and password.

## Notes

- The data is stored in `data.db`.
- `node_modules` and `data.db` are ignored by Git.
